# Code of Conduct

## Our Standard

This project should stay welcoming, practical, and respectful.

Examples of expected behavior:

- be kind and direct
- assume good intent
- give useful technical feedback without insults
- help newer users feel safe asking basic questions
- focus criticism on the code or behavior, not the person

Examples of unacceptable behavior:

- harassment or personal attacks
- hostile, mocking, or demeaning language
- publishing someone else's private information
- intentionally disruptive behavior
- pressure to expose secrets, credentials, or private data

## Scope

This applies to issues, pull requests, discussions, and other project spaces.

## Enforcement

Project maintainers may remove comments, close threads, or block participation for behavior that makes the project unsafe or unwelcoming.

## Reporting

If someone crosses the line, report the behavior privately to the project maintainer instead of escalating publicly.
